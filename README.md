# 🚨 ResQ Link

### Community-Powered Emergency Resource Sharing Platform




\

**Connecting People. Saving Time. Supporting Communities During Emergencies.**

---

## 📖 Overview

ResQ Link is a real-time emergency resource sharing platform designed to connect people with essential resources during crisis situations.

The platform enables communities to quickly share and request:

🩸 Blood Donations

💊 Medicines

🍱 Food Supplies

🚗 Emergency Transportation

🏠 Temporary Shelter

By combining a modern MERN architecture with workflow automation powered by **n8n**, ResQ Link helps improve coordination and reduce delays during emergencies.

## 🔄 How n8n Works in ResQ Link

### Workflow Architecture

```text
User Request
      │
      ▼
React Frontend
      │
      ▼
Node.js Backend
      │
      ▼
n8n Webhook
      │
      ▼
Workflow Processing
      │
      ▼

      │
      ▼
Automated Response
```

## 🚀 Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
cd Backend
npm install

cd ../Frontend
npm install
```

### Run Backend

```bash
npm start
```

### Run Frontend

```bash
npm start
```
